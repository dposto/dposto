## sudo sketchy

Small, focused apps I'm working on.

I've used Linux for several years for personal use. In 2025, I started using it at work. I couldn't find tools for a few things I needed, so I thought I'd make them myself. I'm sharing them with the world because, why not? Jokes aside, I believe in FOSS, GNU/Linux, and the idea that software should be shared and available to anyone who can use it.

---

### Projects

**[Pannex](https://github.com/dposto/pannex)** — *Permissive Annotation Extension*
A screenshot annotation tool geared towards technical writers. Crop, annotate, redact, and publish — without the overhead of a full image editor. Supports step markers, Bézier arrows, highlight and spotlight modes, pixelation, blur, freehand drawing, and FTP/SFTP publishing. Built with Python and PyQt6.

**[Knowledge Article Converter](https://github.com/dposto/ka-converter)**
I doubt anyone outside my workplace would have use for this. It converts exported HTML knowledge base articles into formatted DOCX files. Built to solve a practical documentation problem where LibreOffice couldn't reliably produce presentable results. Uses a modified Pandoc binary, custom Lua filters, and Python for consistent styling. Built with Python and PyQt6.

### Philosophy

- If something's missing, create it.
- Keep things simple. Complexity should be avoided.
- Build for Linux first. Windows if I need to. macOS if someone sends me a Mac.
- Permissive licensing. Use it, modify it, share it.
